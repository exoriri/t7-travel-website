## О проекте

Проект находится в разработке. Далее здесь будет ссылка на рабочий сервис.
Если вы решили посмотреть мой код - смотрите:)) 
Запустить код вы можете, но запросы работать не будут, потому что бэкенд в закрытом репозитории.
В качестве архитектуры используется FSD(Feature-Sliced Design).

## Setup

Make sure to install dependencies:

```bash
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
yarn dev
```
